HTML and CSS Page

HTML:

What exactly is HTML? HTML, stands for Hypertext Markup Language. HTML is the standard form of your everyday web browser. It allows you to change the colors, fonts, font size, background colors, etc. Without it you wouldn't have a basic webpage.

Tags:

Tags in HTML create new sections to your HTML page.

Some tags are : 1) <div></div> -The div tag helps create different sections of your document. 2) <h1></h1> -The h1 tag creates a header. 3) <p></p> -The paragraph tag creates a new tet box in your document. 4) <img src "url"> The image source tag creates an image within your document. To change the size of your picture you can use width and height to size. This can be done by adding width="__" height="__"> after the end quotation mark. 5) <u></u> -The underline tag, underlines text within the tag.

CSS:

What is CSS? CSS is Cascading Style Sheets. CSS is usually used in HTML. In CSS it can be used in order to command certain parts or all of the document. It can change the font, positioning, color of boxes and font,border, image size, font size, width and height of boxes. CSS is great for doing multiple tasks or assigning certain commands to specific elements of the code in HTML. The basic HTML.

Attributes

Atrributes are used in CSS to command certain parts in HTML. All of the following are inserted by an assigned class, div, header,or body. 1) class=___ - The class attribute is one of the most important ones. To use this you have to insert it into your tags in HTML. For example <p class= mentor>____</p> in HTML will result in having to put .mentor into CSS. 2) font-size:__ - This attribute helps create commands or orders to specific elements. 3) background-color:#____; - Changes the background color of a box or the entire page. 4) border:__px - This allows the user to create a thicker or skinnier border for their div. 5) text-align:____; - All text follows directions to be either centered, left, or right.
